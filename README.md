https://jpersonalp.netlify.app/

## :notebook: Bibliotecas usadas
* React vite- v5.1.7;
* React Bootstrap- v2.10.1;
* Sass- v1.71.1;
  
<br>

## :wrench: Instalação

### Instale meu projeto com: 
```bash
  npm install  
```
### Rode ele com:
```bash
  npm run dev  
```
<br>
